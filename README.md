# Lynx-Lang

Lynx-Lang :cat: is an educational project for learning how programming language is made.

## Prerequisites

make sure [rustup](https://rustup.rs/) is pre-installed.

## Build

```sh
cargo build --release
```

## Test

```sh
cargo test
```

## Coverage Test

```sh
sh coverage.sh
```

check `coverage/index.html` in the browser
